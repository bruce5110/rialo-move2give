# Rialo Move2Give

Rialo Move2Give is a concept demo for a fitness challenge service that turns verified workouts into automatic rewards and donations.

The demo highlights:

- Wallet-based challenge participation
- Fitness data connection flow
- Rialo-style private verification
- SFS budget for automation costs
- Automatic winner payout and NGO donation receipts

## Run Locally

Open `index.html` directly in a browser, or run a simple static server:

```bash
python -m http.server 4173
```

Then visit:

```text
http://127.0.0.1:4173
```
